# Database

Database name: website

Import `website.sql` into phpMyAdmin before running the project.

## Steps

1. Open phpMyAdmin.
2. Create a database named `website`.
3. Import `website.sql`.
4. Start Apache and MySQL in XAMPP.
5. Open the project in your browser.
