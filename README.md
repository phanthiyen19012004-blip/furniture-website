# Furniture E-commerce Website

## Project Overview

Furniture E-commerce Website is a PHP & MySQL based web application that allows customers to browse furniture products, manage shopping carts, place orders, and allows administrators to manage products, categories, orders, and coupons.

---

## Technologies

- PHP
- MySQL
- HTML
- CSS3
- Bootstrap
- JavaScript
- XAMPP

---

## Main Features

### Customer

- Register
- Login
- Search Product
- Product Category
- Product Detail
- Shopping Cart
- Checkout
- Order History
- Profile Management

### Administrator

- Dashboard
- Product Management
- Category Management
- Coupon Management
- Order Management
- User Management
